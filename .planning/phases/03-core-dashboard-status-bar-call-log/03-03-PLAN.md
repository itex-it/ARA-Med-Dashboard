From src/lib/hooks/useRealtimeChannel.ts — Realtime hook contract:
- `useRealtimeChannel<T extends { [key: string]: unknown }>({ table, tenantId, events, onChange })`
- onChange receives `RealtimePostgresChangesPayload<T>`; `payload.eventType` is 'INSERT'|'UPDATE'|'DELETE'
- Channel scoped `${table}:${tenantId}`, tenant-filtered, removeChannel cleanup built in

From src/lib/hooks/useOpenTaskCount.ts — the initial-fetch + Realtime-merge pattern to mirror:
- `'use client'`, initial `createClient().from(...).select(...).eq('tenant_id', tenantId)` in useEffect
- Then useRealtimeChannel for live deltas; named row type intersected with `& { [key: string]: unknown }` for the generic

From src/lib/types/index.ts (after 03-01): `CallLogRow` (id, tenant_id, session_id, did_number, phone_hash, duration_seconds, status, intent_main, intent_sub, language_code, summary_short, summary_structured, transcript_text, audio_url, pid_status, patient_recognized, inbox_qualifying, created_at, updated_at, feedback_label, internal_note, intent_corrected) and `CallStatus = 'active'|'completed'|'failed'|'forwarded'|'abandoned'`.
